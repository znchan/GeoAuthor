# GeoAuthor - Supplementary Materials

This file contains supplementary materials for the paper "GeoAuthor: Linking Text and Visualization for Geographic Article Authoring".

## Contents

### 📁 File Structure

```
├── Appendix.pdf                    # Appendix
├── Geographic Article List.xlsx    # List of surveyed geographic articles
├── System Evaluation/              # Evaluation materials
│   ├── Benchmark.json              # Test benchmark dataset
│   └── Results.xlsx                # Evaluation results
├── Video.mp4                       # System demonstration video
└── README.md                       # This file
```

### 📄 File Descriptions

- **Appendix.pdf**: The appendix, including creation examples, a JSON grammar example, a prompt template, and others about the prelininary study and the evaluation.

- **Geographic Article List.xlsx**: A list of geographic articles surveyed in our preliminary study, including article titles and URLs.

- **System Evaluation/**: 
  - `Benchmark.json`: The test benchmark we constructed for evaluating the system's performance in generating geographic visualizations from text
  - `Results.xlsx`: Detailed evaluation results for system evaluation

- **Video.mp4**: Contains a demonstration video showcasing the GeoAuthor system's key features.

The source code for this work, along with all supplementary materials within this directory, is available at https://github.com/znchan/GeoAuthor.
